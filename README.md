# Look weather

Проект позволяет смотреть прогноз погоды выбранного города на ближайшие дни. После выбора города вам будет доступен выбор даты, на которую вы хотите посмотреть прогноз погоды, который отображается на графике.

## Запуск

Для работы проекта необходима установка Redis на ваш компьютер. После этого можете проделать следующие шаги:

- Склонируйте проект себе на компьютер.
- Переименуйте файл "example.properties" в "application.properties", если у вас используется стандартный порт для подключения к redis (6379), то вам ничего менять не нужно.
- Из корня проекта запустите команду ".\gradlew run".
- Готово! Можете пользоваться веб-приложением по адресу "localhost:8080".
